3. 修复方案

修复内容：
✓ 修改了 displayComprehensiveMultiExercise 函数
✓ 添加了字体大小调整控件到标题栏
✓ 将控制按钮移到标题栏中
✓ 与单选模式保持一致的布局
✓ 按钮不再单独占一格
